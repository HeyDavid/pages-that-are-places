# Pages that are Places

A showing of **digital art made through web design** — interactive scenes and signal instruments, built by hand in the browser. Pure static HTML / SVG / Canvas. No build step, no dependencies, nothing to install.

- **№01 — Going Deeper** · scroll from orbit into the abyss; click the asteroids, feed the kelp → `deep.html`
- **№02 — Opportunity Radar** · an animated sonar scope
- **№03 — Signal** · an oscilloscope + tuner

Open `index.html` to walk the show.

## Publish (free, on GitHub Pages)
```bash
# from this folder, in PowerShell:
gh repo create pages-that-are-places --public --source=. --push
# then: repo Settings -> Pages -> Deploy from a branch -> main / (root)
```
Your show goes live at `https://<you>.github.io/pages-that-are-places/` for $0.
